Production plan

Goods in — log a delivery

The traceability code is the goods-in date — the same date written on the sticker. Log each delivery here; staff then pick the lot on the task card when they use it. The invoice ref links the code back to the supplier.

Trace forward — where did a delivery go?

Pick a delivery to see every batch that used it. For a recall: this is the list of affected production.

Trace back — what went into production?

Every batch in the range with its goods-in codes, suppliers and invoice refs.

Recipes (multi-step routes)

Define a finished product as an ordered list of steps (e.g. Brisket = trim → cut → pack → sous-vide → break up → chill). Set a yield % on steps that lose weight, so quantities size correctly down the chain. Auto-plan uses these to schedule the whole chain from a due date.
